Research with Dr. Zoe Boekelheide

This project involves developing data acquisition and organization tools for Dr. Zoe Boekelheide's condensed matter research laboratory in Lafayette College. I was involved in developing central remote control programs for all devices and recording instruments in the lab, creating data piplelines to feed into analytic programs and creating GUIs to help send instructions using few buttons instead of interacting with computer code.

This is my first ever CS project and I am proud to still be working on it :)

Rideshare App Network

This was my third project in Data Structures and Algorithms class in Fall 2019. It involved using graphs to simulate a rideshare network tracking data like:

  • Average trip time during normal and busy hours
  • Average number of customers per driver per day
  • Number of drivers utilized
  • Number of passengers lost
  • Average wait-time of passengers in normal and busy hours

This project involved using primary uses Djikstra's algoritm to calculate shortest path with a changing traffic network environment.

Restaurant Simulator

This was my first project in Data Structures and Algorithms class in Fall 2019. It involved using linked lists and queues to simulate the operation of an eatery on Lafayette College giving data such as:

  • Average customers in normal and busy hours
  • Profitability of the eatery
  • Staff utilization
  • Revenue and cost of operation
  • Satisfaction of customers
  • Average wait-times of customers in normal and busy periods

Random Maze Generator

This project involved creating a graphical representation of mazes used in the IEEE Micromouse competition . This makes it easy for engineers to focus on developing and improving their mice while this tool creates random maps instantly when necessary for testing.

This random maze generator is built in such a way that every cell is accessible from any other cell which is not exactly a requirement for IEEE Micromouse competition. This creates a chance for the mice to truly find the shortest path out of the maximum number of possibilities.

Miner Tycoon

This is the second project for my Data Structures and Algorithms class of Fall 2019. This involved creating a game simulation with the user as the manager of a mine with workers with different abilities on a rectangular map digging for gold while trying to avoid hazards like lava, pits/sinkholes and underground rivers.
